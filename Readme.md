# MATANAT A Website Clone

Bu layihə, MATANAT A veb saytının front-end hissəsinin HTML, CSS və JavaScript istifadə edilərək hazırlanmış klonudur. Layihə əsasən saytın dizaynını və strukturunu təkrar qurmağa yönəlib.

---

## 🚀 Layihə haqqında

Bu layihə çoxsəhifəli (multi-page) statik veb saytdır və real korporativ sayt strukturuna uyğun şəkildə hazırlanıb. Heç bir framework istifadə edilmədən yazılıb ki, əsas front-end bacarıqları inkişaf etdirilsin.

---

## 🛠️ İstifadə olunan texnologiyalar

* HTML5 (semantic struktur)
* CSS3 (Flexbox, responsive dizayn)
* JavaScript (Vanilla JS)

---
